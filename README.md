# Vanilla Nodejs RestFul CRUD API Project | No Framework or Library
> This is Node.js CRUD API movie project.

## Routes
```bash
GET      /api/movies
POST     /api/movies
GET      /api/movies/:id
PUT      /api/movies/:id
DELETE   /api/movies/:id

```


## Installation

```bash
# Install dependencies
npm install



# Run in production
npm start
```
